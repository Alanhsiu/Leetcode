This track is a practical tour of the **GCP services you reach for most often**,
with the mental model for each and the `gcloud`/`bq` commands to get started.

Because these are infrastructure commands that act on real cloud projects, the
code blocks here are **read-only references with a copy button** — they are not
executed in the browser (faking that would be misleading). Use them as a
copy-paste starting point against your own project.

Confirm flags, names, and pricing against the
[official Google Cloud docs](https://cloud.google.com/docs).

### Services covered
- **Cloud Run** — serverless containers.
- **GKE** — managed Kubernetes.
- **Pub/Sub** — asynchronous messaging.
- **Cloud Storage** — object storage.
- **IAM** — who can do what on which resource.
- **BigQuery** — serverless data warehouse.
